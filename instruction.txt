============================================================
      USER GUIDE: DYNAMIC MIKU PLYMOUTH SYSTEM
============================================================

This system provides a high-definition, 24 FPS animated boot 
splash that automatically rotates its routine every day.

--- HOW IT WORKS ---

1. THE POOL: All available animation frames are stored in 
   `/opt/MikuPlymouth/miku_plymouth_clip_pool/`.
2. THE ROTATION: The `install.sh` script picks 10 random clips 
   from this pool and copies them into the active theme.
3. THE GENERATOR: It then writes a custom Plymouth script 
   that shuffles those 10 clips and pre-loads them into RAM 
   to ensure zero lag during boot.
4. THE HOOK: Finally, it rebuilds your system's boot image 
   (`initramfs`) so the new routine is ready for the next boot.

--- SYSTEM LIMITATIONS ---

- WHY ONLY 10 CLIPS? 
  Plymouth runs in your computer's RAM before the main OS 
  is loaded. 1080p images are heavy. 10 clips (240 frames) 
  requires ~2GB of RAM to play smoothly. Using more clips 
  can cause the boot to freeze or the screen to stay black.
- INITRAMFS SIZE: 
  Every clip you add increases your boot image size by 
  ~20MB. Keeping it to 10 clips ensures your system stays 
  fast and healthy.

--- SETTING UP DAILY ROTATION (AUTOMATION) ---

The system includes a systemd timer that runs 10 seconds 
after every boot to refresh the Miku routine for the next day.

1. Install the automation files:
   ```bash
   sudo cp /opt/MikuPlymouth/miku-rotate.service /etc/systemd/system/
   sudo cp /opt/MikuPlymouth/miku-rotate.timer /etc/systemd/system/
   ```
2. Activate the timer:
   ```bash
   sudo systemctl daemon-reload
   sudo systemctl enable --now miku-rotate.timer
   ```
3. Check the status:
   ```bash
   systemctl list-timers miku-rotate.timer
   ```

--- CUSTOMIZING YOUR ROTATION ---

If you want to pick specific clips you love:

METHOD 1: THE "POOL PRUNING" (Recommended)
Simply go to `/opt/MikuPlymouth/miku_plymouth_clip_pool/` and 
delete the folders or frames for clips you dislike. 
The next time the script runs, it will only pick from your 
favorites.

METHOD 2: THE "EXACT SELECTION"
If you want exactly specific clips every time:
1. Open `/opt/MikuPlymouth/install.sh`.
2. Find the line: `selected_clips=$(echo "$available_clips" | shuf -n 10 | sort)`
3. Change it to: `selected_clips="clip002 clip015 clip021"` 
   (using your preferred clip numbers).
4. Run `./install.sh` manually to apply.

--- NVIDIA & SYSTEM REQUIREMENTS ---

For this theme to work, your system must support Early KMS.
- GRUB: `/etc/default/grub` must contain `quiet splash nvidia-drm.modeset=1`.
- HOOKS: `/etc/mkinitcpio.conf` must have `nvidia` modules in the `MODULES` array.
- The `install.sh` script attempts to fix these automatically for you.

--- SUMMARY OF COMMANDS ---

- Manual Refresh: `sudo /opt/MikuPlymouth/install.sh`
- View Logs: `journalctl -u miku-rotate.service`
- Test Preview: 
  `xhost +local:root && sudo plymouthd --no-daemon --x11 & sleep 2 && sudo plymouth --show-splash`
